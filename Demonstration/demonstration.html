Demonstration

This page is the simulator to the actual hardware. This was going to be drag and drop but because of time constraints this did not happen. This is the simplest way that we came up with and that would work to simulate the hardware. Have fun with the trial!

Instructions

This may seem pretty straight forward but it might confuse some people. In order to effectively simulate the hardware, you will need to select any of the IDs that are listed. PLease do not choose the same ID more than once. The buttons are used to complete the simulation either by scanning the RFID card/tag or by going to the dashboard. Order of clicking does not matter, but you may want to open the dashboard (by clicking the button) first and then view the data update as you scan the card. The data should show up in the dashboard and you should be done with the simulation. Please make sure to use all IDs for best result. Have fun!
