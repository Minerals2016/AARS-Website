What Is AARS?

Automatic Attendance Recording System (AARS) is a system complex system that uses both hardware and software to automate the task of taking attendance as fast and as efficient as possible. AARS uses a Raspberry Pi and a RFID detector along with RFID cards to count attendance. It allows teachers and organizations to save time from manually counting attendance and allows organizations to manage workers shifts better. AARS is both a payed and a free service. We offer the hardware needed for $79.99 USD per (class)room while we offer our software for free. Our software is what separates us from our competition. This is where we give our customers the best experience they can have. Make sure to try the demo to see if AARS is right for you and your organization.

Features and Specifications

The project was designed with a school setting in mind, but it may also be used on an office setting. We know that our project might have already been widely implemented for security (such as HID systems) but we wanted to make a widescale and cheap alternative; AARS does exactly just that. Each sensor (1 unit) will cost around $70 USD compared to $200 usd average of industry grade controllers. The software on most industry grade systems is also paid separately, compared to AARS which uses free and open source software. This project relies more on the software than hardware so you know that you will not lose much if anything happens to any sensor, it can get replaced and set up with our software at no additional charge.
